Open in New Tab

Given no extra attributes, the anchor will open the link in the current tab. To open in a new tab, add the target="_blank" attribute.

Link to open in new tab

Target Security Risks

Adding the target attribute poses a security risk, as JavaScript can modify the original page. To prevent this, add rel="noopener noreferrer" attribute.

"noopener" prevents JavaScript from modifying the original page. "noreferrer" does not tell the new tab where the user came from.

Types of Links

Absolute Links

Absolute links have been used above where they include the scheme and domain of the resource. These links often link out to external websites.

Relative Links

Just like using the command line, using "." and ".." refer to the current and parent directories, respectively, on the local machine.

So if I have another html file within the folder that this index.html is in, I do not need to use dot notation.

But if I want to link back to the "lists" lesson, in another folder, I need to use dot notation by using ../lists/index.html

Images

We can use previous knowledge of relative/absolute links to also display images using the <img> tag with the src attribute

Always include an alt attribute for users with screen readers and a replacement for if the image doesn't load.

Image Sizing

To prevent the webpage from jumping and flashing while loading, specify an image size (even if modified by CSS later) using the height and width attributes
